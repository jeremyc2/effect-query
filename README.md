# effect-query

A vibe fork of [TanStack Query](https://github.com/TanStack/query), written in [Effect V4](https://github.com/Effect-TS/effect-smol), making heavy use of [Tim Smart](https://github.com/tim-smart)'s Effect Atom, which I believe has now been folded into Effect V4 itself and expanded upon.

To install dependencies:

```bash
bun install
```
